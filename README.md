# 🌐 Personal Portfolio – Nicolás Barragán

> Backend & Systems-oriented Fullstack Developer
> Rust • TypeScript • Microservices • DevOps

📄 También disponible en Español: [README.es.md](./README.es.md)

This repository contains the source code for my personal web portfolio, developed as an academic project for both my Computer Science and Cybersecurity undergraduate programs.

The purpose of this project is to showcase my professional profile, technical stack, and software development experience through a modern, lightweight web application designed with internationalization support.

---

## 🚀 Features

* Professional profile presentation (#perfil)
* Technologies overview (#tecnologias)
* Projects section (#proyectos)
* Contact form (#contacto)
* Multi-language support (English / Spanish)
* Responsive interface
* Component-based architecture
* Content internationalization (i18n)

---

## 📦 Installation & Usage

Clone the repository:

```bash
git clone https://github.com/Nicobgn/nicobgn.github.io.git
```

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm run dev
```
