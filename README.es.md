# 🌐 Personal Portfolio – Nicolás Barragán
> Backend & Systems-oriented Fullstack Developer  
> Rust • TypeScript • Microservices • DevOps

Portfolio web personal desarrollado como proyecto académico para la Licenciatura en Informática y Licenciatura en Ciberseguridad.

El objetivo de este proyecto es presentar mi perfil profesional, stack tecnológico y experiencia en desarrollo de software mediante una aplicación web moderna, ligera y preparada para entornos internacionales.

---

## 🚀 Características

* Presentación del perfil profesional (#perfil)
* Listado de tecnologías utilizadas (#tecnologias)
* Sección de proyectos (#proyectos)
* Formulario de contacto (#contacto)
* Soporte multi-idioma (Inglés / Español)
* Interfaz responsive
* Arquitectura basada en componentes
* Internacionalización (i18n) mediante diccionario de contenidos

## 📦 Instalación y Uso

Clonar el repositorio:

```bash
git clone https://github.com/Nicobgn/nicobgn.github.io.git
```

Instalar dependencias:

```bash
pnpm install
```

Ejecutar en modo desarrollo:

```bash
pnpm run dev
```
